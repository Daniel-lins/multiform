# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/Daniel-lins/multiform
```

```bash
# Install Dependencies
$ npm install

# Run Aplication
$ npm start
```
Go to http://localhost:3000/ to see the result.
